☁️ クラウド同期の設定（Firebase）

スマホとPCで履歴を自動同期するには、無料のFirebaseを使います。
Googleアカウントがあれば5分で設定できます。

1
Firebaseコンソールを開く
https://console.firebase.google.com
Googleアカウントでログインしてください。
2
新しいプロジェクトを作成
「プロジェクトを追加」→ 名前は何でもOK（例: architect-quiz）
Googleアナリティクスは「無効」でOK
3
Firestoreデータベースを作成
左メニュー「Firestore Database」→「データベースの作成」
「テストモードで開始」を選択 → 「有効にする」
4
ウェブアプリを追加してConfig取得
左上のプロジェクト概要 → 「</>（ウェブ）」アイコン → アプリ名を入力して登録
表示される firebaseConfig = { ... } の
{ } の中身だけをコピーして下の欄に貼り付けてください。

☁️ クラウド同期：接続中

解答を採点するたびに自動でクラウドに保存されます。
スマホでこのページ（GitHub Pages）を開いて、同じ学習IDを入力すれば同期されます。

🔑 学習ID（デバイス間共有キー）

スマホとPCで同じ履歴を共有するための合言葉です。
覚えやすい文字列を設定してください（例: tanaka2025 など）。
※ 他の人が同じIDを使うと履歴が混在するので、ユニークな文字列を推奨します。
